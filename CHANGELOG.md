# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-02-05

### Added
- Initial release
- Zone model for 6 geopolitical zones
- State model for 36 states + FCT
- LGA model for 774 Local Government Areas
- City model for cities and towns
- Ward model for electoral/administrative wards
- PostalCode model for postal codes
- Django admin integration for all models
- Management command `load_ng_locations` to populate database
- Comprehensive utility functions for location queries
- Complete Nigerian geographic data (zones, states, LGAs)
- Full documentation and usage examples
- PyPI package configuration

### Features
- Search and filter capabilities in Django admin
- Helper functions for common location queries
- Support for Django 4.0+ and Python 3.8+
- Reusable across Django projects
